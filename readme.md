## 商户查询缓存
（1）更新缓存保证数据一致性最好采用先更新数据库，再删除缓存。这样可以使发生数据不一致的情况降到最低。

（2）添加事物注解使二者同步更新。

（3）缓存穿透两种解决办法：缓存空对象、布隆过滤。
